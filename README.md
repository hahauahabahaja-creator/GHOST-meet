# GHOST meet

Premium, cloud-based automated meeting capture suite.

## Features
- **1080p HD Capture**: Full-screen browser recording.
- **Visual Overlay**: Premium dark-themed dashboard.
- **Telegram Control**: `/join`, `/record`, `/stop`.
- **Hybrid Economy**: Resource-efficient stream management.
- **Transcription**: Free Hindi + English transcription.
- **Auto-Chunking**: Splits recordings into 40MB chunks for Telegram.
